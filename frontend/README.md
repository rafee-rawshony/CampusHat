# CampusHat Frontend

This is the React + Tailwind frontend of CampusHat.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Axios

## Setup

```bash
npm install
npm run dev
```
---

Folder Structure
```bash
src/
 ├─ components/
 ├─ pages/
 ├─ layouts/
 ├─ services/
 ├─ hooks/
```
Rules

All API calls must be inside services/
eusable UI in components/
Pages only for routing
